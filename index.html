Autocomplete - Table

One major challenge we faced in autocomplete was to display the data in a tabular format. When there are many rows of information available, it was difficult to display the relevant data of a row next to each other. This forced to develop a JQuery plugin which can display the relevant information in a table format.

The initialization is simple as like any other plugin

var myData = $("#mytextbox").tautocomplete();

Parameters

The plugin has following parameters,

Table width, example:

width: specify the table width

Table columns. The first column is always the ID. This should not be mentioned in the parameter, example: 

columns: ['Country', 'Code', 'Capital']

Callback function is triggered once a selection is made, example:

callback: function () {alert("Selected ID is : text2.id()); }

Message to be displayed when no records found, example: 

norecord: "No Records Found"

Data Property for the textbox can also be mentioned, example:

dataproperty: {"mandatory":"yes", "property": "something"}

Allowed inputs in the textbox can be mentioned using Regex, default is:

regex: "^[a-zA-Z0-9\b]+$"

A callback function which passes the information to be displayed in the autocomplete table. In the below example, the textbox is used to search the countries. The function filters the data and passes the relevant information to be shown in the autocomplete-table.

data: function () {
    var data = [
                { "id": 1, "country": "Afghanistan", "code": "AFG", "capital": "Kabul" }, 
                { "id": 2, "country": "Albania", "code": "ALB", "capital": "Tirane" },
                { "id": 11, "country": "Azerbaijan", "code": "AZE", "capital": "Baku" }, 
                { "id": 12, "country": "The Bahamas", "code": "THA", "capital": "Nassau" }];
    var filterData = [];
    var searchData = eval("/" + text2.searchdata() + "/gi");
    $.each(data, function(i,v)
    {
      if (v.country.search(new RegExp(searchData)) != -1) {filterData.push(v);}});
      return filterData;
    }

Properties

The following Properties are available for the plugin

id: returns the ID of the selected row

text2.id()

text: returns the TEXT of the selected row

text2.text()

searchdata: returns the text which user is trying to search

text2.searchdata()

isNull: returns true if no selection is made

text2.isNull()
